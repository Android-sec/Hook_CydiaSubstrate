# cydiasubstrate_hooktools
An android hook tools based on cydia substrate

QQgroup：456853837

## 描述
在 Eclipse 中可以正常对 java/Jni 函数进行 开发/编译/安装

## 运行
在手机中只需要安装 `cydiasubstrate_hooktools.apk` 和 `cydiasubstrate apk` 就可以了。
